Z ZENURA — Usage Guide
Overview
Eye-controlled alert dashboard
ZENURA allows a paralysed or speech-impaired patient to call for help using only eye movements and blinks. Electrodes on the face detect tiny electrical signals when eyes move — no hands or voice needed.
The 4 Action Tiles
NURSE
Sends an immediate Telegram alert to the assigned doctor or nurse. Use when medical attention is needed.
FAMILY
Notifies designated family contacts via Telegram. Use when you want to reach loved ones.
LIGHTS
Toggles the physical room LED light on ESP32 GPIO 26. No Telegram sent — hardware control only.
SOS
Critical emergency broadcast sent to doctor, family, and friend simultaneously. Use only in real emergencies.
EOG Mode — Eye Control
STEP 1 — DIRECTION Look LEFT → selects left column (NURSE / LIGHTS) Look RIGHT → selects right column (FAMILY / SOS) Look CENTER → enters REST zone — nothing triggers STEP 2 — SELECT Single blink → activates TOP tile in your column Double blink → activates BOTTOM tile in your column Triple blink → PAUSE / RESUME system (freezes all tiles) Stare 1 second → dwell-activates the tile under the cursor
Demo Mode — Mouse Control
1
Move mouse over any tile — it highlights with a glow border
2
Click to trigger immediately — or hover for 1 second to auto-trigger via dwell
3
A Telegram message is sent and a voice alert plays automatically
4
Use the SWITCH TO EOG button in the header to switch modes
Electrode Placement
Red wire → Right temple (outer eye corner) → RA on AD8232 Yellow wire → Left temple (outer eye corner) → LA on AD8232 Green wire → Center forehead (reference) → RL on AD8232 Press each pad firmly. Keep face still during 3-second calibration on power-on. If ELECTRODE DETACHMENT appears — press the pads firmly again. The Electrode Sensor widget in the sidebar shows live contact status.
Chrome voice fix: Click Enable Voice in the sidebar on first load. This is a one-time browser permission requirement.
